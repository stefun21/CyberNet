## 🎮 Core Gameplay & Mechanics

### 1. The Hacking Core & Thermodynamic Engine
* **Manual Data Extraction:** Click the central server node to inject structural data packets and harvest **ByteCoins (BC)**.
* **Dynamic Heat Grid:** Every manual extraction generates heat. Push the core too hard, and you'll trigger a **Thermal Emergency (Overheat Lockdown)**. 
* **Ultra-Fluid 60FPS Cooling:** Equipped with a high-fidelity rendering pipeline (`requestAnimationFrame`) that dissipates core temperature at a blinding rate of **66.6% per second**, whether idling or recovering from a system crash.

### 2. Hardware & Software Upgrades
Invest your mined BC into 6 deep layers of network infrastructure. The local shop instantly tracks your financial status, automatically graying out (`disabled` visual node state) modules you cannot afford:
* `Syringe_Injector.click` — Boosts manual packet data mining value.
* `Infiltration_Script.sh` — Deploys automated network scripts.
* `Plutonium_GPU_Rig.dat` — Harnesses nuclear power to calculate massive hash arrays.
* `Orbital_Mainframe_Hijack` — Reroutes satellite traffic directly to your wallet.
* `AI_Darknet_Hivemind.net` — Leverages next-gen model weights for automated logging.
* `Dyson_Sphere_Data_Core` — Captures entire stellar energy clusters for infinite crypto scaling.

### 3. Cosmic Prestige System (Quantum Singularity)
Collapse your current timeline. Convert your massive ledger capital into **Quantum Chips**, resetting your financial balance but unlocking a permanent, compounding global multiplier (`+12%` efficiency per chip) for all future runs.

### 4. Endgame: Mastery Achievement Reset
The terminal tracks **15 core decrypting milestones** (from triggering your first meltdown to capturing ancient anomalies). 
* Once all 15 achievements are unlocked, the terminal surfaces the high-level **`ACTIVATE MASTERY RESET`** protocol.
* Activating this clears your current achievements, unlocks **Gold Mastery status**, and scales all future target requirements to keep the simulation challenging.
* In exchange for this ultimate loop reset, your mainframe receives a **permanent boost of +5 free levels across every hardware and software upgrade module** instantly injected into the active runtime database.

### 5. Reality Glitch Anomalies
Keep your eyes on the terminal workspace. Volatile code anomalies will randomly spawn across your monitor space:
* 🔥 **Red Core:** Accelerates manual extraction to a 4x multiplier for 15 seconds.
* 🌀 **Blue Core:** Overclocks the Glitch Engine, forcing manual clicks to yield 12x values plus a 6% extraction bonus based on your passive network generation.
* 🪙 **Gold Core:** Instantly captures a massive high-crypto financial ledger payout.

## 🛠️ Architecture & Tech Stack

* **Frontend Engine:** Vanilla HTML5 / ES6 Javascript (No heavy third-party dependencies for maximum terminal processing speed).
* **Styling Framework:** Custom CSS3 layout styled with a retro-futuristic monochrome matrix theme, crt-overlay animations, and shaking shake effect classes.
* **State Management:** Fully integrated persistent local database sync hooks via browser `localStorage` ensuring your mined assets are archived between sessions.

## 🚀 Deployment

The interface is optimized for rapid cloud delivery. To spin up a local instance of the mainframe terminal:

1. Clone this repository to your developer environment.
2. Ensure `index.html`, `style.css`, and `script.js` are in the same structural directory layer.
3. Open `index.html` in any web browser, or connect the repository to **Vercel / GitHub Pages** for instantaneous global edge hosting.
